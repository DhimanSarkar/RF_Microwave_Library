Microwave Toolbox

  • Calibration
    • uW_cal_define_standard — define the set of parameters for a SOLT standard such as C0 for the "open" standard.
    • uW_cal_offset2delay — Scattering parameter matrix to Cascade matrix convertion for 2 ports matrix.
    • uW_cal_standard — generate the S1P form of a standard according the definition parameters.
  • De-embedding
    • uW_AMCAD_CST_deembedding — Load-pull measurement de-embedding .
    • uW_S2P_deembedding — S2P measurements de-embedding .
  • Display
    • uW_MES_plot — Plot graph from IV measurements curves (MES file)
    • uW_S2P_display — Display either S1P or S2P scilab object in the current graphic window as a VNA.
    • uW_S2P_stab_display — Plot the stability circles from S2P data.
    • uW_display_smith — Plot the smith chart in the current graphic window.
    • uW_display_vsmith — Plot the smith chart in the current graphic window with vectorial arcs.
  • MES files
    • uW_MES2CITI — Generate a CITIfile from a MES file (*.mes) from XLIM measurements
    • uW_MES_get_quiescent — quiescent point informations for a selected IV measurement file (*.mes) network
    • uW_MES_plot — Plot graph from IV measurements curves (MES file)
    • uW_MES_read — read IV measurement file (*.mes) from XLIM measurements
    • uW_MES_select_network — Display a dialog box to select a measured network
    • uW_MPS_read — read a S2P from a multiple S parameter file (MPS)
  • S2P files
    • uW_CSV_read — Read Sij data from a CSV file.
    • uW_S1P_read — Read data from a S1P file.
    • uW_S1P_write — Write S data into a S1P file.
    • uW_S2H — Scattering parameter matrix to Hybrid matrix convertion for 2 ports matrix.
    • uW_S2P_DELTA — Calculate the determinant of an S2P file.
    • uW_S2P_K — Calculate the stability K factor.
    • uW_S2P_MAG — Calculate the Maximum Available Gain (MAG) of a S2P.
    • uW_S2P_MSG — Calculate the Maximum Stable Gain (MSG) of a S2P.
    • uW_S2P_deembedding — S2P measurements de-embedding .
    • uW_S2P_display — Display either S1P or S2P scilab object in the current graphic window as a VNA.
    • uW_S2P_flip — Flip port 1 and port 2 of a S2P object.
    • uW_S2P_freq — Write S data into a S2P file.
    • uW_S2P_read — Read data from a S2P file.
    • uW_S2P_split — Split a line into two parts.
    • uW_S2P_stab — Returns the centers and radius of input stability circles for a S2P.
    • uW_S2P_stab_display — Plot the stability circles from S2P data.
    • uW_S2P_to_SnP — Create a SnP object from several S2P measurements or datas.
    • uW_S2P_u1 — Calculate the Edwards-Sinsky stability parameter : µ1 of a S2P.
    • uW_S2P_u2 — Calculate the Edwards-Sinsky stability parameter : µ2 of a S2P.
    • uW_S2P_unoise — Cancel noise on a S2P measurement
    • uW_S2P_write — Write S data into a S2P file.
    • uW_S2T — Scattering parameter matrix to Cascade matrix convertion for 2 ports matrix.
    • uW_S2Y — Scattering parameter matrix to Admittance matrix convertion for 2 ports matrix.
    • uW_S2Z — Scattering parameter matrix to Impedance matrix convertion for 2 ports matrix.
    • uW_SnP_write — Write SnP data into a SnP file.
    • uW_T2S — Cascade parameter matrix to Scattering matrix convertion for 2 ports matrix.
  • CITIfile
    • uW_CST2CITI — Convert time domain load-pull CST measurement file into a CITIfile for Agilent ADS.
    • uW_MES2CITI — Generate a CITIfile from a MES file (*.mes) from XLIM measurements
    • uW_mS2P2CITI — Generate a CITIfile from a S2P directory
  • Other
    • uW_freq_unit — Convert frequency unit in scaling factor to get a frequency in Hertz.
    • uW_unwarp — Unwarp the phase of a Sij parameter
